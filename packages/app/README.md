A simple App to manage Flutter SDK Versions

FVM helps with the need for a consistent app builds by allowing to reference Flutter SDK version used on a per-project basis. It also allows you to have multiple Flutter versions installed to quickly validate and test upcoming Flutter releases with your apps, without waiting for Flutter installation every time.

## Features

- Install and remove Flutter SDK releases & channels
- Manage stored Flutter SDK versions
- Change Flutter SDK per project
- Search all channels, releases, and projects.
- View details about each Flutter release
- Remove unused Flutter SDK Versions

## Windows
- [Turn on developer mode](https://docs.microsoft.com/en-us/windows/uwp/get-started/enable-your-device-for-development)  to properly link versions to projects or run in Administrator mode.
